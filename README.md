# Trackify - Modern Job Application Tracker

Trackify is a lightweight, local-first web application designed to help job seekers organize, log, and optimize their career search pipeline. Built with a minimal, high-performance user interface, it provides data insights and status columns out of the box.

## Key Features Implemented
* **Dynamic Pipeline Analytics:** Live dashboard breakdowns tracking conversion metrics across applications.
* **Custom Metric Logging:** Configurable fields to track specific application criteria.
* **Persistent Browser Memory:** Built-in lightweight state management using modern client-side storage architecture.
* **Theming:** Full native dark/light mode toggle supporting a custom high-visibility accent color configuration.

##  Tech Stack & Architecture
* **Frontend Core:** React 18, TypeScript
* **State Management:** Redux Toolkit / Context API
* **Styling Foundation:** Custom SCSS design tokens with Tailwind utility classes
* **Build Engine:** Vite
